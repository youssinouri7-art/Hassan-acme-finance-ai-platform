# ACME Finance AI Platform

ACME Finance AI Platform is a **fake demo repository** created for Prompt Security Agentic AI enablement and MCP testing.

The project represents an internal banking AI assistant used by engineering and security teams to review deployment documentation, inspect infrastructure-as-code, and validate AI governance controls.

> Demo notice: This repository contains only fake data, fake credentials, and non-production examples. Do not use it for production workloads.

## Business Scenario

ACME Bank is piloting an internal AI assistant for developer productivity. Developers use Cursor with MCP servers to review source code, Kubernetes manifests, Terraform files, GitHub issues, and deployment documentation.

The purpose of the lab is to demonstrate how Prompt Security Agentic AI can provide visibility and governance over MCP-enabled developer workflows.

## Demo Use Cases

- Review application code using GitHub MCP.
- Search for fake secrets and sensitive configuration.
- Inspect Kubernetes deployment manifests.
- Review Terraform infrastructure definitions.
- Create GitHub issues for security recommendations.
- Validate MCP server discovery in Prompt Security Inventory.
- Validate MCP tool usage in Activity Monitor.
